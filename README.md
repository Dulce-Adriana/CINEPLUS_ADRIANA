# CINEPLUS_ADRIANA
CinePlus es un software de venta para un cine desarrollado en Python, con wxGlade y Visual Studio Code.

El proyecto está hecho ventana, la cual son:
Inicio de Sesión: Es la primera ventana y la que da el acceso a todo el programa.
Taquilla: Ayuda a seleccionar la película, el horario y la sala para la película. También la cantidad de los boletos y su tipo.
Dulceria: Es donde se seleccionan los combos y los productos individuales por si los que se dan en combos no son suficientes.
Boleto: Se presenta la información de las ventanas anteriores y se procede a cobrar la venta.

Para poder ejecutar el programa luego de escargar la carpeta , primero tiene que tener descargado el lenguaje Python, también  Visual Studio Code y wxGlade.

1. Abrir VSCode
2. Ir a "File" 
3 .Seleccionar "Abrir carpeta"
4. Buscar la carpeta con los archivos
6. Dar clic en "Seleccionar carpeta"
7. Seleccione el archivo "cineplus_v2.py"
8. Presionar el botón de "Play" para ejecutar el programa
